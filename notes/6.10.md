## Contention Resolution

设备知道 n -> let p = 1/n \
成功的概率 > 1/e, n 趋于无穷时等于 (与时隙ALOHA计算方法一样) -> 泊松分布 \
有限 n -> 近似泊松分布

Transmission Rule:
- P(silent) = $e^{-p_i}$
- P(transmit) = $p_ie^{-p_i}$ -> halt
- P(noise) = $1 - (1 + p_i)e^{-p_i}$ (unnecessary)

$\downarrow$

Channel Feedbacks:
- $p_{sil} = \prod_{i\in [n]} e^{-p_{i}} = e^{-c}$
- $p_{suc} = \sum_{i\in[n]}p_{i}e^{-p_{i}}\cdot\prod_{j\in[n]\backslash\{i\}}e^{-p_{j}}=ce^{-c}$
- $p_{noi} = 1-(1+c)e^{-c}$

聚合冲突度 aggregate contention: $c = \sum_{i\in[n]} p_{i}$

c = 1 时 $p_{suc}$ 最大 -> 设计 Update Rule 的目标: 使 c 向 1 变化 \
-> c = 1 时: $p_{sil} = 1/e \neq p_{noi} = 1 - 2/e$ & 更新规则应无偏 -> 对 sil 和 noi 的更新应成比例

$\downarrow$

Update Rule:
$$p_{i}\leftarrow\begin{cases}p_{i}\cdot e^{\epsilon} & \text{听到静默时} \\ p_{i} & \text{听到成功时} \\ p_{i}\cdot e^{-\epsilon/(e-2)} & \text{听到噪音时}\end{cases}$$

步长 step size $\epsilon > 0$, 算法唯一的参数

概率: 乘性增加 $\rightarrow$ 衡量冲突度 c: 对数尺度 $\rightarrow$ $\gamma = \ln(c)$

未发送成功时，$\gamma$ 随机游走 \
设 $\gamma$ 和 $\gamma^{\prime}$ 分别是更新前后的值，则 $\gamma^{\prime}\in\{\gamma-\frac{\epsilon}{e-2},\gamma,\gamma+\epsilon\}$ \
将 $\gamma$ 处的吸引力 attraction 定义为 $\gamma^{\prime}-\gamma$ 的期望(以步长 $\epsilon$ 为单位): $attr(\gamma)=p_{sil}(\gamma)-\frac{1}{e-2}\cdot p_{noi}(\gamma)=e^{-e^{\gamma}}-\frac{1}{e-2}\cdot\left(1-(1+e^{\gamma})e^{-e^{\gamma}}\right)$ \
即 $\mathbb{E}[\gamma^{\prime}]=\gamma+\epsilon\cdot attr(\gamma)$ \
当 $\gamma\rightarrow-\infty$ 时 $attr(\gamma)$ 渐近于 1，而当 $\gamma\rightarrow\infty$ 时渐近于 $-1/(e-2)$ 

注: 不直接处理 $attr(\gamma)$ 的实际表达式, 使用一个分段线性近似: 
$\hat{attr}(\gamma)=\begin{cases}3/5 & \gamma<-1 \\ -(3/5)\gamma & \gamma\in[-1,1] \\ -3/5 & \gamma>1\end{cases}$ \
$attr(0)=\hat{attr}(0)=0$; 当 $\gamma\ne0$ 时，$attr(\gamma)/\hat{attr}(\gamma)\ge1$, 即预测值绝对值小于真实值，是保守预测

该游走对原点具有一定的正向吸引力

### 2.1 Interlude: Homesick Random Walks 插曲：趋心随机游走

趋心 - 随机游走表现出向原点的吸引力 \
简化版随机游走的平稳分布 (stationary distribution)

形式化: \
整数 $[-\delta^{-1},...,\delta^{-1}]$（其中 $\delta>0$）上的随机游走 \
第 $t$ 步位于 $\pm i$, 第 $t+1$ 步:
- 以概率 $(1+i\delta)/2$ 向 0 移动 (即移动到 $\pm(i-1)$)
- 以概率 $(1-i\delta)/2$ 远离 0 (即移动到 $\pm(i+1)$)

位于 0 时, 等概率移动至 +1 / -1

设 $\pi(i)$ 为平稳分布下处于 $i$ 或 $-i$ 的概率, 则有: \
$\pi(0)=\pi(1)\cdot\frac{1+\delta}{2}$ \
$\pi(i-1)\cdot\frac{1-(i-1)\delta}{2}=\pi(i)\cdot\frac{1+i\delta}{2}$ \
[$\text{从 } i-1 \text{ 流向 } i \text{ 的概率总量} = \text{从 } i \text{ 流回 } i-1 \text{ 的概率总量}$]
$$\pi(i)=2\pi(0)\cdot\prod_{j=1}^{i}\frac{1-(j-1)\delta}{1+j\delta}=2\pi(0)\cdot\prod_{j=0}^{i-1}\frac{1-j\delta}{1+(i-j)\delta}>2\pi(0)(1-i\delta)^{i}\approx2\pi(0)e^{-i^{2}\delta}$$
即: $\pi$ 的常数比例的质量都集中在区间 $[-\sqrt{\delta^{-1}},\sqrt{\delta^{-1}}]$ 内

### 2.2 The Efficiency Curve 效率曲线

当 $\gamma$ 偏离最优值时，效率是如何退化的 - 由其在 $\gamma\in[-\sqrt{\epsilon},\sqrt{\epsilon}]$ 极端位置的行为决定

对 $p_{suc}(\gamma)=e^{\gamma}e^{-e^{\gamma}}$ 在 $\gamma=0$ 处进行泰勒展开的前几项得到近似:
$$p_{suc}(\gamma)=\frac{1}{e}-\frac{\gamma^{2}}{2e}-\frac{\gamma^{3}}{6e}+O(\gamma^{4})>\frac{1}{e}-\gamma^{2}\left(\frac{1}{2e}+\frac{1}{6e}\right)>1/e-\gamma^{2}/4$$

预期 $\gamma$ 将有常数比例的时间处于 $[-\sqrt{\epsilon},\sqrt{\epsilon}]$ 区间内, 而在此区间内, 该算法的预期通道利用率至少为 $1/e-\epsilon/4$
